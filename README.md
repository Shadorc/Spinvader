<h1>Spinvader</h1>

<h2>Introduction</h2>
Spinvader is a Java 2D game that incorporates the basics of the legendary game Space Invader by adding some features.
It is still in the development phase.

<h2>Control :</h2>
<ul>
	<li>Movement : ZQSD</li>
	<li>Shoot : Space</li>
	<li>Show Hitbox : F4</li>
	<li>Debug Mod : F3</li>
</ul>
